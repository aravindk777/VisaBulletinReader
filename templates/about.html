About Visa Bulletin Reader

Before this, many of us navigate to the official US Travel.state.gov website to check the latest updates. I felt myself many times. It's not just me, but many other aspiring Developers. So, why not a web app that can read and display the latest visa bulletin dates... and that's how this application was built.

It all started as a learning project about web scraping and why not build a purposeful one.

{#

Goal

#} {#

The goal of this project is to provide a user-friendly way to access visa bulletin information from the official US government website. It started as a learning project to explore Python and web scraping capabilities.

#}

Featuring,

  • Scrapes the latest "Upcoming" or "Current" Visa Bulletin from travel.state.gov.
  • Allows the users to filter by Visa Type (Family or Employment) and Country.
  • Displays "Dates For Filing" and "Final Action Dates" in a clean table format.
